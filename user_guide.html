一、系统概述

智能绿建评价系统是一个专业的绿色建筑评估工具，可帮助您根据不同标准（如成都市标、四川省标或国标）对建筑项目进行绿色建筑评价。系统支持项目信息管理、条文评分、生成评估报告等功能。

🔔 系统亮点：本系统支持多种评价标准，能够灵活适应不同地区不同类型的绿色建筑评价需求，为设计、施工和评估提供全面支持。

二、登录与注册

1. 登录系统

  • 在浏览器中输入系统网址：acbim.cn
  • 在登录页面输入您的用户名和密码
  • 点击"登录"按钮进入系统

2. 注册账号

  • 如果您是新用户，请点击登录页面的"注册"链接
  • 填写必要的个人信息和邀请码（需要管理员提供）
  • 点击"注册"按钮完成账号创建

3. 忘记密码

  • 如果忘记密码，请点击登录页面的"忘记密码"链接
  • 按照页面提示进行密码重置操作

三、项目管理

1. 创建新项目

  • 登录后，进入"项目管理"页面
  • 点击"新建项目"按钮，系统将弹出项目创建对话框
  • 填写项目基本信息，包括：
    • 项目名称（必填）
    • 评价标准（必填，选择"成都市标"、"四川省标"或"国标"）
    • 建筑类型（必填，选择"居住建筑"、"公共建筑"或"居住+公共建筑"）
    • 目标星级（必填，选择"基本级"、"一星级"、"二星级"或"三星级"）
  • 点击"创建项目"按钮完成项目创建

2. 查看项目列表

  • 在"项目管理"页面可以查看您创建的所有项目
  • 可以通过搜索框快速查找项目
  • 点击项目名称进入项目详情页

3. 编辑项目信息

  • 在项目详情页点击"项目信息"选项
  • 修改项目相关信息，如项目名称、项目编号、建设单位、设计单位、项目地点等
  • 点击"保存项目信息"按钮保存修改

4. 删除项目

  • 在项目列表中找到要删除的项目
  • 点击项目行末尾的"删除"按钮
  • 确认删除操作

⚠️ 注意：项目删除后将无法恢复，请谨慎操作！

四、标准评分

1. 基本级评分

系统提供以下专业的基本级评分：

  • 建筑
  • 结构
  • 给排水
  • 电气
  • 暖通
  • 景观
  • 环境健康与节能（仅四川省标）

操作步骤：

  • 进入项目详情页面
  • 在左侧菜单中选择"基本级"下的相应专业
  • 系统将显示该专业下所有条文
  • 对每条条文进行评分：
    • 选择"是否达标"
    • 输入具体得分
    • 填写技术措施（可选）
  • 评分会实时保存

2. 提高级评分

系统提供与基本级相同专业的提高级评分，操作步骤与基本级相同：

  • 在左侧菜单中选择"提高级"下的相应专业
  • 按条文进行评分
  • 系统自动保存评分结果

3. 计算评分

完成各专业评分后，您可以计算项目总评分：

  • 点击页面上的"计算评分"按钮
  • 系统将自动计算并更新以下分数：
    • 各专业分数
    • 各章节分数（安全耐久、健康舒适、生活便利、资源节约、环境宜居、提高与创新）
    • 项目总分
    • 评定结果（几星级）

💡 提示：建议在完成所有条文评分后再进行计算评分，以获得最准确的结果。

五、报告导出

1. 导出报审表

  • 在左侧菜单中选择"报告导出"
  • 在展开的子菜单中点击"报审表"
  • 系统将根据项目评分数据生成报审表
  • 生成完成后会自动提供下载链接

2. 导出绿色建筑设计专篇

  • 在"报告导出"菜单中选择"绿色建筑设计专篇"选项
  • 系统将根据项目的评价标准和星级目标选择相应的DWG模板
  • 将项目信息和评分数据填充到DWG文件中
  • 生成完成后可以下载DWG文件
  • 下载的DWG文件可以使用AutoCAD打开查看和编辑

3. 导出绿色建材应用比例

  • 在"报告导出"菜单中选择"绿色建材应用比例"选项
  • 系统将显示绿色建材应用比例计算页面
  • 根据需要输入相关数据和参数
  • 点击"导出Word"按钮生成计算书
  • 可以下载Word格式的绿色建材应用比例计算书

4. 导出绿建自评估报告（国标项目专用）

  • 当项目的评价标准为"国标"时，在"报告导出"菜单中会显示"绿建自评估报告"选项
  • 点击"绿建自评估报告"选项
  • 系统将根据项目评分数据生成自评估报告
  • 生成完成后可以下载Word格式的报告文件

六、界面导航

1. 主菜单

  • 项目信息：显示和编辑项目基本信息
  • 基本级：展开后显示基本级各专业评分选项
  • 提高级：展开后显示提高级各专业评分选项
  • 评分汇总：显示各专业和章节的评分汇总情况
  • 报告导出：展开后显示各种导出选项（报审表、绿色建筑设计专篇、绿色建材应用比例等）

2. 用户菜单

  • 点击右上角的用户图标可以打开用户菜单
  • 菜单选项包括：
    • 个人信息
    • 修改密码
    • 返回项目列表
    • 退出登录

七、常见问题

1. 分数不正确怎么办？

  • 检查是否有遗漏未评分的条文
  • 点击"计算评分"按钮重新计算
  • 如问题仍存在，请联系系统管理员

2. 无法保存评分怎么办？

  • 检查网络连接是否正常
  • 刷新页面后重试
  • 确认您有该项目的编辑权限

3. 报告无法导出怎么办？

  • 确保已完成所有必要条文的评分
  • 检查网络连接和浏览器兼容性
  • 等待一段时间后重试
  • 如果是导出DWG文件，请确保进度条显示完成

八、注意事项

  1. 系统会定期自动保存您的评分数据，但建议在重要更改后手动保存
  2. 完成所有条文评分后才能得到准确的项目总分
  3. 导出报告前请确保项目信息完整准确
  4. 为保证系统安全，长时间不操作将自动退出登录
  5. 使用结束后请点击"退出登录"安全退出系统
